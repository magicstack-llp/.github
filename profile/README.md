# MagicStack Enterprise LLP

## About Us

MagicStack is a software development and consulting company based in Toronto, Canada, founded in 2008. We specialize in building business analytics software and supporting the entire business development cycle with cutting-edge technology solutions.

## Our Expertise

We build applications and backends using:
- **Python** - Core development language with contributions to the Python ecosystem
- **PostgreSQL** - Advanced database systems and optimization
- **JavaScript** - Modern, reactive web development

## Services

- **Custom Software Development** - Enterprise solutions across ERP systems, document management, billing/accounting, and e-commerce platforms
- **Cloud Infrastructure & DevOps** - Scalable cloud implementations with security and compliance focus
- **AI & Automation** - AI-powered chatbots and intelligent automation solutions
- **Technical Consulting** - Tackling challenging technical problems across the entire software lifecycle

## Our Flagship Product

### EdgeDB
EdgeDB is our modern object database system that blends the strengths of relational databases with object-oriented features. Born from years of experience building advanced software systems, EdgeDB represents the next generation of database technology.

## Open Source Contributions
